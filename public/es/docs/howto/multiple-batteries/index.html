Soporte para Multiple Baterías

Muestra el estado de las baterías por instancia.

Por defecto el estado de la batería primaria va a ser mostrado en la barra i3-bar. Para algunas laptops que tienen múltiples baterías, por ejemplo ThinkPad, vas a querer ver ambas baterías. Para hacerlo, sigue los pasos:

  1. Monta tu propio archivo de configuración i3 (tutorial), y luego abre~/.config/regolith/i3xrocks/conf.d/80_battery en el editor de tu preferencia.
  2. Encuentra la línea que corresponde al blocket de la batería:
...
# Battery indicator
#
# Displays total percentage charge left of specified battery.
[battery]
interval=30
# instance=BAT0
  1. Lista todas las instancias de batería corriendo en tu maquina ejecutando:
ls /sys/class/power_supply/

# AC BAT0 BAT1
  1. Actualiza el archivo de configuración montado para que tenga un blocket nuevo como el siguiente:
...
# Battery indicator
#
# Displays total percentage charge left of specified battery.
[battery]
interval=30
instance=BAT0

[battery]
interval=30
instance=BAT1
