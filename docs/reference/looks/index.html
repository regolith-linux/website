Regolith Looks

A gallery of visual themes (called looks) provided by Regolith

Regolith provides several visual styles, known as “looks”, that can be installed and used. To keep the install size of Regolith low, these looks are installed as Debian packages. By default only one look is shipped with Regolith: Lascaille. Other looks may be installed via the package manager and then loaded via super alt l .
